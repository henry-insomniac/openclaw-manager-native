# OpenClaw Manager Native 1.0.4

发布日期：2026-03-11

## 这版改了什么

- 诊断中心继续收短，改成“先做这一步 / 最近一次操作 / 维护建议”的结构，小白用户可以直接用它维护本地 OpenClaw。
- 原生壳继续收口 AppKit 主线程路径，降低 reopen、菜单栏和弹窗相关的崩溃风险。
- daemon 做了性能优化：`manager` 并发读取 profile 并缓存短期 usage，`support/summary` 改成并发采集状态、配置、Gateway 和环境信息。
- 安装版实测：`/api/openclaw/manager` 从约 `8.9s` 降到约 `0.5s`，`/api/support/summary` 从约 `12.0s` 降到约 `3.5s`。
