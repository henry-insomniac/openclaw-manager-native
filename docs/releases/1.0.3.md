# OpenClaw Manager Native 1.0.3

发布日期：2026-03-11

## 这版改了什么

- 原生 app 文案继续收短，控制台、账号池、诊断和命令页只保留状态、动作和必要说明。
- provider-aware 第一阶段落地：daemon 开始读取 `openclaw.json` 的 primary provider / primary model，非 Codex profile 不再因为没有 `.codex` 凭据直接判死。
- 默认激活同步从“只同步 Codex”扩展到“同步整个 OpenClaw auth/config”，Codex 保留为可选 companion runtime。
